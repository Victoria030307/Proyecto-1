Proyecto 1: Análisis de Redes Sociales

Este proyecto desarrolla una aplicación para el análisis de redes sociales (modeladas como grafos dirigidos), enfocada en la detección y visualización de Componentes Fuertemente Conectados (CFC) usando el algoritmo de Kosaraju con DFS. La aplicación permite cargar, modificar y visualizar el grafo, identificando los grupos de usuarios (CFC) en la red.
